# Changelog

## v0.1.0-alpha-2 (code 100) — 30.06.2026

### Новое
- **Измерение АЧХ**: модуль измерения частотной характеристики (14 типов тестовых сигналов, Welch periodogram, EqFitter)
- **Генератор тестовых сигналов**: розовый шум (Kellet + FFT), белый, коричневый, синий, фиолетовый, серый шум; лог. и линейный свип; ступенчатый тон; multi-tone; MLS; Dirac-импульс
- **Полная русская локализация**: values-ru/strings.xml (460+ строк), заменены 42 хардкодные строки на ресурсы
- **Выбор типа сигнала** в интерфейсе замера АЧХ (Spinner с 14 вариантами)

### Исправления
- **Critical BUG 1.1**: Graphic/Table контроллеры не перестраивались после загрузки пресета из пикера — добавлены вызовы `buildSliders()`/`buildTable()`
- **Bug 1.2**: `updateSliderValues()` в `GraphicEqController` мог вылететь с IndexOutOfBounds при изменении количества полос — добавлена проверка с авто-перестроением
- **Локализация**: заменены хардкодные строки ON/OFF/L/R, статусные тексты, backup/restore сообщения, контент-дескрипшены

### Технический долг
- Добавлены 55 новых строковых ключей в strings.xml
- DRY-рефакторинг: хардкодные строки заменены на getString() в 5 Kotlin-файлах
- Создан Spinner с адаптером ArrayAdapter для выбора типа сигнала

## v0.1.0-alpha-1 — 29.06.2026

### Исправления (10 проблем по результатам аудита кода)

- **ProGuard**: blanket `-keep class com.bearinmind.equalizer314.**` заменён на точечные правила
- **DynamicsProcessing fallback**: добавлен перебор приоритетов (Int.MAX_VALUE-1 → 100k → 10k → 1k) для OEM-совместимости
- **Watchdog backoff**: интервал проверки сессии 0 изменён с фиксированного 5s на экспоненциальный (15s..60s)
- **Валидация пресетов**: добавлена проверка имён (blank, reserved `__`-префикс)
- **EqSerializer.Keys**: JSON-ключи вынесены в `object Keys` (11 констант)
- **ChannelMath**: защита от ArrayIndexOutOfBounds при несовпадающих размерах массивов
- **Locale.US**: исправлены 89 вызовов `String.format()` в 14 файлах
- **softClip**: замена `tanh(buffer[i].toDouble()).toFloat()` на кубический soft-clamp (Float-only, ~2x быстрее)
- **Thread-safe bandColors**: `mutableMapOf` → `ConcurrentHashMap`
- **runCatching с логированием**: пустые `catch (_: Exception) {}` заменены на `runCatching {}.onFailure { Log.w(...) }`

## v0.0.12-beta — 17.06.2026

### Новое
- **Светлая и тёмная тема**: переключатель Light Theme в Настройках (вместо старого Simple-EQ). Мгновенная перерисовка всего приложения — экранов, графика, измерителей. Выбор сохраняется после перезапуска.
- **Резервное копирование и восстановление**: экспорт всех настроек, пресетов и привязок устройств/приложений в единый .json-файл. Импорт на чистой установке или при смене источника APK (issue #37).
- **Перезапись пресетов**: кнопка Overwrite для каждого сохранённого пресета — быстрая перезапись текущего EQ в существующий пресет (issue #45).
- **Per-band fill кривые**: вклад каждой полосы отображается полупрозрачной цветной кривой под суммарной белой линией (стиль Allen & Heath SQ-5 / FabFilter) (issue #40).
- **Simple-режим использует общий пул пресетов**: пресеты Simple-режима теперь хранятся вместе с Parametric/Graphic/Table и отображаются в едином списке (issue #39).
- **Фильтры RBJ**: Bell-фильтры переведены с Vicanek на RBJ — равные противоположные пики корректно обнуляются без остаточного ripple (issue #41).
- **L/R-разделение при автопереключении**: пресеты, привязанные к устройствам и приложениям, теперь корректно применяют независимые фильтры для левого и правого каналов (issue #46).
- **Watchdog сессии 0**: при захвате глобальной аудиосессии другим приложением EQ автоматически переподключается (issue #47).
- **Настройки реорганизованы**: AutoEQ & Presets, Generate Custom EQ, Convert to APO объединены в новый экран «Presets & Conversions». Ползунки Parametric dB расширены до ±20 дБ.

### Исправления
- Иконки в заголовке графика больше не деформируются при смене темы.
- Исправлен запуск MediaPlayback на устройствах Pixel (issue #38).
- Кнопки Reset и EQ ON/OFF корректно расположены; ON/OFF плавно скрывается при появлении Reset.
- Простой режим для главного экрана — исправлено ломающееся XML для кнопок графика.
