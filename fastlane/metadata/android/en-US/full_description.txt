Equalizer314 is a system-wide parametric equalizer for Android with full DynamicsProcessing API support (127 bands), multiband compression, limiting, reverb, and real-time audio-visual feedback via the Visualizer API.

Unlike other EQ apps, Equalizer314 provides a complete audio-visual feedback loop — see exactly what each band, compressor, and limiter does in real time. All features are free and unlocked; nothing is premium-locked.

✨ WHAT'S NEW
- Frequency Response Measurement: measure your headphones/speakers using 14 test signal types (pink noise, white noise, log sweep, MLS, Dirac impulse, and more). The Welch periodogram analyzes the captured signal and automatically creates a correction EQ preset.
- Russian localization: the entire interface is now available in Russian (460+ translated strings).

🎛️ FEATURES
- 4 EQ modes: Parametric, Graphic (31 bands), Table, Simple (7 bands)
- 127-band DSP via DynamicsProcessing API
- Independent Left/Right channel EQ (Channel Side EQ)
- Multiband compressor (MBC, 1-6 bands) with soft-knee, GR trace, Linkwitz-Riley crossover
- Limiter with waveform visualization and LUFS metering
- Environmental Reverb (10 parameters)
- AutoEQ import: Squiglink, Wavelet, EqualizerAPO formats
- Preset binding to Bluetooth devices and apps
- Undo/Redo for all EQ modes
- FFT spectrum analyzer with multiple display modes
- APO import/export, Wavelet/Poweramp EQ conversion
- Full backup & restore (single .json file)
- Foreground service with boot persistence
- Quick Settings tile
- Light & Dark theme
- Detect and auto-apply presets per output device

🔒 PRIVACY
- No internet access (INTERNET permission is NOT declared)
- No analytics, no ads, no trackers, no accounts
- Only on-device processing
- Open source under GPL-3.0

This app is built off of DynamicsProcessing API; this is the same API that Poweramp EQ & Wavelet both use. Using the Visualizer API in tandem gives you an intimate audio-visual feedback loop that puts you in complete control of your audio.
